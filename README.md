# Global Happenings

Realtime happenings on the world globe demo. Right-click to change globe state.

Three.js / D3 / GSAP

[lempiy.github.io/global-happenings](https://lempiy.github.io/global-happenings)

### Installation

```
npm install
```

### Start Dev Server

```
npm start
```

### Build Prod Version

```
npm run build
```
